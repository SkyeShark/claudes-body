Claude Face Atlas Builder

Renders the eye, brow, and mouth shapes from renderer/character.js into texture atlases for 3D facial UV mapping. Each atlas is a horizontal strip of cells, one cell per shape variant. The 3D shader picks a cell by setting a single integer uniform and offsetting the UV. All three atlases share the same coordinate system (viewBox -64 -64 128 128), so cells line up when stacked or sampled together.
